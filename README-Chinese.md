
go-datastructures
=================

go-datastructures是一系列有用, 高性能且线程安全的Go数据结构. 

### 注: 仅在Go 1.3+版本测试过.

#### Augmented Tree - 增强树

用于n维区间冲撞的区间树. 通过红黑增强树来实现. 超出的维度由同时的 插入/查询 来处理, 
这样虽然使得复杂度不是最佳的, 但可以节省空间. 利用bit数组进行插入探测. 在单维度内,
插入, 删除以及查询, 应该在O(log n)的时间复杂度内. 

#### Bitarray - Bit数组

无需借助于散列和哈希图, Bit数组用于探测存在性. 需要实体对象拥有一个uint64类型的唯一
标识符. 存在两种实现, 常规的和稀疏的. 稀疏的Bit数组可以节省可观的空间, 但是插入的复
杂度是O(log n). Bit数组提供的接口, 有一些有用的方法来探测两个Bit数组之间的交叉. 

#### Futures - 

发送"广播"消息给监听者的一个有用的工具. 通道存在这样的问题, 一个监听者从一个通道中
取出一条信息, 其他监听者并不知情. 很多情况下, 当我们想要将单一事件通知给多数监听者
时, Futures包便能提供帮助.

#### Queue - 队列

Queue包提供了普通队列和优先队列. 两者的实现, 都不会阻塞发送且能根据所需拓展大小. 尝
试向一个设置队列插入值的话, 两者都只会返回一个错误; 此外, 向一个已关闭的通道发送消息
